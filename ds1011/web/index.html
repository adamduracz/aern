    • overview of various setup steps for use in Aston labs
      • Start Eclipse 3.6 with EE support - see Practical 3 (22 Oct)
      • (more to come)
    • running your own translation service
      • these instructions work only for a recent Ubuntu; you can possibly use Ubuntu on a small virtual machine
      • to install all required software, connect your computer to the Internet and type the following in a terminal:
        sudo apt-get install sun-java6-jdk apertium-en-es apertium-en-ca
      • extract in your home cwk-translator-binary.zip
      • execute in terminal:
        cd cwk-translator-binary; sh run
      • test your translation service installation by opening in a Web browser the URL:
        http://localhost:3000/langPairs
        if you see "es:en en:es ca:en en:ca" the service is up and running
