CS3250 09/10 Distributed Systems: how to use the startDS, et al scripts

  • Some programs are very slow in the labs due to the central filesystem.
    To speed it up, log in to Ubuntu and use the following programs:
    startDS, saveDS, stopDS
    to create, save and remove a folder CS3250-DS-0910 that appears in you home but is in fact on the local computer's disk.
    detailed instructions:
    • the following steps need to be carried out only once initially:
      • download the above programs and save them in your home folder (not Desktop)
      • open a terminal and execute: chmod +x *DS
      • if you already have a folder called CS3250-DS-0910, rename it, eg to CS3250-DS-0910-OLD
      • execute: ./startDS (this should create the CS3250-DS-0910 temporary folder)
      • start eclipseEE with the workspace CS3250-DS-0910/workspace instead of the default one
      • execute firefox -p and create a new profile
        in the folder CS3250-DS-0910/firefox,
        which you create within the select folder dialogue
      • move your previous DS work into the CS3250-DS-0910 folder
      • ensure you have at least 50MB free space
        (to get a nice tool to do this, press Alt-F2 and type baobab and press Enter, then click "analyse home")
      • execute: ./stopDS (this backs up and removes the CS3250-DS-0910 folder)
    • the following steps need to be carried out in each lab session:
      • login to Ubuntu
      • execute: ./startDS (this should create the CS3250-DS-0910 temporary folder)
      • extract any given DS projects inside CS3250-DS-0910
      • open firefox, eclipse, etc and work on given tasks
      • whenever you want to backup your work, execute ./saveDS
      • stop firefox and eclipse
      • ensure you have at least 50MB free space
        (to get a nice tool to do this, press Alt-F2 and type baobab and press Enter, then click "analyse home")
      • execute: ./stopDS (this backs up and removes the CS3250-DS-0910 folder)
        if you omit this step, you are likely to lose your data!
      • press Ctrl-Alt-Del and click Shutdown
